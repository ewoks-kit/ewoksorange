Installation
============

.. tab-set::

    .. tab-item:: from pypi

        .. code-block:: bash

            pip install ewoksorange[full]

    .. tab-item:: from source code

        .. code-block:: bash
            
            git clone https://github.com/ewoks-kit/ewoksorange.git
            # or cloning with ssh
            # git clone git@github.com:ewoks-kit/ewoksorange.git
            cd ewoksorange
            pip install .[full]
