===========
ewoksorange
===========

*ewoksorange* provides a desktop graphical interface for `ewoks <https://ewoks.readthedocs.io/>`_
based on `biolab <https://github.com/biolab>`_ libraries `orange-canvas-core <https://github.com/biolab/orange-canvas-core>`_
and `orange-base-widget <https://github.com/biolab/orange-widget-base>`_.

*ewoksorange* is developed by the `Software group <http://www.esrf.eu/Instrumentation/software>`_
of the `European Synchrotron <https://www.esrf.eu/>`_.
